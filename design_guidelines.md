# Design Guidelines for 홈클린리치 (Home Clean Rich) Website

## Project Overview
Single-page cleaning service website replicating the exact structure of https://cleanalert.imweb.me/ with all content customized for 홈클린리치.

## Color Scheme
- **Background/Spacing**: White (#FFFFFF) - primary color for clean, fresh feel
- **Accent Color**: Naver-green (bright green #03C75A) for:
  - CTA buttons
  - Icons and emphasis elements
  - Section dividers
  - Highlighted text
  - Navigation active states

## Exact Section Structure (11 Sections)

### 1. Header/Navigation
- 홈클린리치 logo (customer-provided)
- Menu items: 입주청소, 거주청소, 상가청소, 특수청소
- Contact/consultation CTA button with 홈클린리치 contact information
- Fixed/sticky navigation on scroll

### 2. Hero/Main Banner
- Headline: "입주청소의 새로운 기준, 홈클린리치!"
- Value proposition copy
- Primary CTA buttons for consultation
- Replace all "깨끗주의보" mentions with "홈클린리치"
- Background image (customer-provided)

### 3. Statistics Section
Three key metrics with animated counters:
- 청소 완료 건수: [X]+ 건
- 고객 만족도: [X]%
- 하루 문의건수: [X]건

### 4. Three Key Advantages
Cards with icons (customer images) and descriptions:
1. **전문 청소** - 꼼꼼한 청소 CHECK
2. **맞춤형 서비스** - 고객 맞춤 CHECK
3. **신뢰할 수 있는 청소** - 신뢰도 CHECK

### 5. Four Service Cards
Grid layout with images and "자세히 보기→" buttons:
- **입주청소** - 입주청소 · 이사청소
- **거주청소** - 거주청소 · 정기청소
- **상가청소** - 상가청소 · 사무실청소
- **특수청소** - 특수청소 · 화재복구

### 6. Before & After Gallery
- Section title: "Before & After 홈클린리치 작업 사례"
- Customer-provided before/after photos in grid layout

### 7. Review Section
- Section title: "Review 홈클린리치의 솔직리뷰"
- Customer testimonial cards layout

### 8. 8-Step Service Process
Sequential workflow visualization:
1. 문의접수
2. 온라인 상담
3. 견적제안
4. 계약체결
5. 일정확정
6. 전문팀 방문 (홈클린리치 전문가 팀)
7. 청소작업 (홈클린리치 매뉴얼)
8. 작업완료 (홈클린리치 품질 검수)

Replace all "깨끗주의보" with "홈클린리치"

### 9. Live Service Booking Status
Real-time booking notifications:
- "김**님 서비스 예약이 접수되었습니다." + "접수완료" badge
- Scrolling/animated list of recent bookings

### 10. Contact Form
Form fields:
- 이름 (Name)
- 연락처 (Phone)
- 시공 선택 (Service selection checkboxes)
- 내용 (Message)
- Submit button with green accent

### 11. Footer
Company information (to be updated with actual 홈클린리치 data):
- Company name
- Representative
- Location
- Phone number
- Email
- Business registration number
- Copyright notice

## Typography
- Clean, modern Korean font (Noto Sans KR or Pretendard)
- Hierarchy: Large hero headlines → Section titles → Body text
- Font weights: Bold for headings, Regular for body

## Layout & Spacing
- Consistent vertical rhythm with generous white space
- Centered content with max-width containers
- Section padding: py-16 to py-24
- Card grids: 3 columns (advantages), 4 columns (services) on desktop

## Images
- All images replaced with customer-provided 홈클린리치 materials
- Logo, work photos, before/after shots
- Hero section uses full-width background image
- Service cards with representative images

## Interactive Elements
- Smooth scroll navigation between sections
- CTA buttons with green background, hover states
- Animated counter for statistics
- "자세히 보기→" buttons on service cards
- Mobile-responsive hamburger menu

## Responsive Design
- Mobile-first approach
- Stack columns on mobile
- Maintain all sections and content hierarchy
- Touch-friendly button sizes
